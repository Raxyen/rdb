# rdb
## Raxyen Debugger
My own debugger made in C for Linux operating systems as my Engineering Thesis on Computer science degree on NCU Toruń.

## Implemented
- CPU registers (RAX, RBX, ...)
- Stack values starting of stack pointer
- Live view

## In progress
- Disassembly tool for most common x86-64 instructions
- More aesthetic text UI
